Get started in 60 seconds

Three steps — no install, no account, no configuration files to edit.

🍎 macOS
🪟 Windows
1
Download & unzip
Download the zip above, unzip it anywhere — your Desktop, Documents, wherever you like.
2
Run setup in Terminal
Open Terminal, cd into the folder and run bash setup.sh. It will ask you to create a PIN, generate a TLS certificate, then start the server. The terminal prints your iPhone URL.
3
Open on iPhone (same WiFi)
Open the printed URL (e.g. https://192.168.1.x:5001) in Safari. First visit: tap Show Details → Visit This Website to trust the local cert. Optionally add to Home Screen via the Share button.
1
Download & unzip
Download the zip above, right-click → Extract All. Put the folder anywhere you like.
2
Double-click setup.bat
Double-click setup.bat. An interactive menu lets you choose which features to enable. It checks for Python, creates your PIN, generates a TLS cert, then starts the server. Your Android URL is printed in the window.
3
Open on Android (same WiFi)
Open the printed URL in Chrome on Android. First visit: tap Advanced → Proceed to trust the local cert. Add to Home Screen via Chrome's menu → Add to Home screen for a full-screen app feel.

🔒 Security

🏠
Local network onlyServer binds to port 5001 on your LAN. Nothing leaves your home network.
🔐
HTTPS + self-signed certAll traffic is encrypted. A unique TLS certificate is generated on your machine on first run.
🔑
PBKDF2-SHA256 PINYour PIN is hashed with 200,000 rounds. The raw PIN is never stored anywhere.

🛡 Privacy

☁
Zero cloudNo analytics, no telemetry, no servers, no accounts. Everything runs on your machine.
🔓
Open sourceEvery line of code is on GitHub. Read it, audit it, fork it.
🚫
Rate limiting5 failed PIN attempts triggers a 5-minute lockout per IP address.
